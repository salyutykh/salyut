# Персональный сайт Сергея Лютых

Статический адаптивный сайт без CMS и внешних библиотек. Его можно разместить на любом хостинге, который раздаёт HTML-файлы.

## Состав

- `index.html` — содержание сайта;
- `styles.css` — дизайн и адаптивная верстка;
- `script.js` — мобильное меню, фильтр публикаций и анимация;
- `assets/` — портрет и иконка сайта;
- `404.html` — страница ошибки;
- `robots.txt`, `sitemap.xml` — базовая SEO-настройка.

## Быстрый просмотр на компьютере

В папке сайта выполните:

```bash
python -m http.server 8080
```

Затем откройте `http://localhost:8080`.

## Публикация

### Самый простой вариант: GitHub Pages

1. Создайте репозиторий на GitHub.
2. Загрузите в корень все файлы из этой папки.
3. Откройте Settings → Pages.
4. Выберите публикацию из ветки `main`, папка `/root`.
5. После появления технического адреса подключите домен в разделе Custom domain.

### Вариант с удобным автодеплоем: Cloudflare Pages или Netlify

1. Подключите GitHub-репозиторий.
2. Build command оставьте пустой.
3. Output directory укажите `/`.
4. Добавьте собственный домен в настройках проекта.
5. Сервис покажет DNS-записи, которые нужно добавить у регистратора домена.

## Что нужно заменить перед запуском

1. Проверить и зарегистрировать подходящий домен.
2. В `index.html`, `robots.txt` и `sitemap.xml` заменить `sergeylyutykh.ru`, если выбран другой адрес.
3. При желании заменить портрет `assets/sergey-lyutykh.jpg` на фотографию высокого разрешения с тем же именем.
4. Проверить контактную почту.
5. Добавить Telegram, VK или другие публичные ссылки в подвал и блок контактов.

## Обновление публикаций

Карточки находятся в секции `id="publications"` файла `index.html`. Для новой карточки скопируйте блок `<article class="story-card">`, замените дату, заголовок, описание и ссылку. Категории фильтра задаются в `data-category`, например:

```html
data-category="north sport"
```

Доступные значения: `north`, `sport`, `travel`.
